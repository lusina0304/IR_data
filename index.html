복합리조트 지능형 서베일런스 공공데이터 서비스

현 공공 데이터는 문화체육관광부의 “복합리조트(Integrated Resort) 산업의 안전을 위한 AI 학습용 데이터 구축 및 지능형 서베일런스 플랫폼 기술개발” 과제 수행의 일환으로 2단계에 취득한 서베일런스 학습용 데이터를 공개하는 것임.

  1. 과제 개요
  • 본 연구 과제는 복합리조트 산업에서 증대되는 고객 관리, 고객 보호, 영업장 보호의 요구에 대응하기 위하여 인공지능 기반 지능형 서베일런스 플랫폼을 개발하고 취득된 학습데이터의 공공화를 통해 국내 서베일런스 시장의 경쟁력을 강화시키는 목표를 가짐
  • 데이터 취득 및 관리, 활용이 가능한 지능형 서베일런스 플랫폼 개발
  • 취득된 데이터와 레이블링을 통한 학습데이터 구축
  • 얼굴 인식을 통한 고객 관리 기술 개발
  • 상황 인식을 통한 고객 보호 기술 개발
  • 테이블 게임의 게임 도구(카드, 칩) 및 손 상호작용 인식을 통한 영업장 보호 기술 개발
  • 학습데이터 및 AI 모델 공개를 통한 복합리조트 지능형 서베일런스 기술 공공화

Model Web Api 통신

 

1.     서버에 LOGIN을 요청하면 클라이언트는 API KEY 수신

2.     서버에 Predict을 요청하면 클라이언트-서버 간 Websocket 연결

3.     서버는 Predict 결과(JSON 형식)를 실시간으로 클라이언트에 전송

4.   
서버에 LOGOUT을 요청하면 Websocket 연결이 종료
